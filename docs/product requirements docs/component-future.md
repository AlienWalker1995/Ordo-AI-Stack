## How to Add a New Component
1. Create a new file under `product requirements docs/` (e.g., `component-xxxx.md`).
2. Add an entry to `index.md` under the `Component Docs` list.
3. Document its purpose, responsibilities, API, and integration points.
4. Reference the component in the overall PRD.
